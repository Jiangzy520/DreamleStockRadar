# 论坛配图清单

发论坛时，建议搭配下面 4 张图。这样读者点进仓库后更容易理解你分享的重点。

## 图 1：首页总览

建议内容：
- 首页 Hero 区
- 概览卡片
- 多源输入说明

建议说明文案：

`公开版首页，主要展示项目定位和系统结构。`

## 图 2：网站模块说明

建议内容：
- 多源输入
- Dashboard
- 股票模拟盘
- 期货模拟盘

建议说明文案：

`公开版保留了网站形态和系统模块说明，真实策略已移除。`

## 图 3：架构说明

建议内容：
- 多源输入
- 统一接口层
- 展示层
- 模拟盘执行层

建议说明文案：

`论坛分享重点是系统架构和产品化过程，而不是直接暴露策略实现。`

## 图 4：GitHub 仓库页

建议内容：
- README 顶部
- docs 目录
- 运行方式

建议说明文案：

`仓库中提供了论坛配套说明文档，方便读者继续查看。`

## 发布前检查

- 页面上不要出现真实服务器 IP
- 页面上不要出现第三方私有接口名
- 截图里不要出现账号、Token、日志明细
- 如果有演示数据，也尽量使用通用占位文案
